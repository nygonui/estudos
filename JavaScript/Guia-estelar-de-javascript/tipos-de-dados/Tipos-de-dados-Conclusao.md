# Tipos de dados

Conforme o ECCMAScript standard temos 9 tipos de dados:

* Data types 
  * Primitive / Primitive value
  * Structural
  * Structural Primitive

## Primitivos 

* String 
* Number
* Boolean
* undefined
* Symbol
* GigInt

## Estruturais

* Object
  * Array
  * Map
  * Set
  * Date
  * ...

## Primitivo Estrutural / Structural Root Primitive

* null
